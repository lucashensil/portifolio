Análise de Dados de Anime

Este projeto de análise de dados visa responder a várias perguntas sobre animes. Foram abordadas questões como: quais gêneros são mais e menos comuns, estatísticas dos scores, animes nos extremos de score, correlação entre número de episódios e popularidade, se os animes com maior score são os mais populares, quais gêneros têm a maior pontuação média e a distribuição dos animes por classificação etária.
